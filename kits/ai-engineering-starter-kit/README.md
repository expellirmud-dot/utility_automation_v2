# AI Engineering Starter Kit

Reusable bootstrap kit for AI-assisted software projects.

Includes:
- AGENTS.md
- GEMINI.md
- Serena memory files
- reusable prompt templates

Purpose:
- reduce repeated prompt context
- align AI agents with project rules
- prevent architecture drift
- support repeatable code/test/report workflows

Usage:
1. Copy this folder into a new project.
2. Move AGENTS.md and GEMINI.md to the project root.
3. Copy prompts/ to the project root.
4. Copy .serena/memories/ if using Serena.
5. Edit rules to match the new project.
