# Tool Matrix

Last Updated: 2026-05-20

## Objective
This matrix categorizes the available Serena and repository-local AI tools by functionality and authority boundaries.

## Symbolic & Search Tools (Serena)
* `find_symbol`: Retrieve symbol definitions by name path.
* `get_symbols_overview`: Overview of file structure and exports.
* `find_referencing_symbols`: Dependency analysis and impact assessment.
* `search_for_pattern`: Heuristic text searches across workspace.

## File & Memory Operations (Serena)
* `read_file` / `create_text_file`: Targeted file access.
* `replace_content`: Regex-based contiguous substitutions.
* `read_memory` / `write_memory`: Operational semantic memory access.

## Validation Tools
* `python -m pytest`: Test framework.
* `deterministic_certifier`: Governance enforcement gate.
