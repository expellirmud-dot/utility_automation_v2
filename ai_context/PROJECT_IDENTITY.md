# Project Identity

## Project
utility_automation_v2

## Core Mission
Build a deterministic governance-driven autonomous operations platform.

This is NOT an unconstrained autonomous AI executor.

Primary characteristics:
- deterministic
- auditable
- governed
- certifiable
- replayable
- operationally inspectable
- human supervised
- authority bounded

## Philosophy
Autonomous intelligence may:
- analyze
- classify
- simulate
- recommend
- diagnose
- review

Autonomous intelligence may NOT:
- mutate governed truth autonomously
- bypass governance
- silently execute authority actions
- override certification

## Intended Platform Shape
A controlled governance operations platform, not a freeform agent playground.
