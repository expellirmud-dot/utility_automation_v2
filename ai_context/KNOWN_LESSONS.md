# Known Lessons

## Chat Context Fragility
Long ChatGPT conversations preserve nuance but can become slow.

Important project memory must move into repository documents.

## Read-First Works
Mandatory READ-FIRST reduces hallucination and unsafe edits.

## Project-Local Override Matters
Global agent setup is useful, but project rules must win.

## Gemini CLI Gotchas
- wrong workspace blocks file access
- model switch may fail or require entitlement
- pasted prompt can collapse as "Pasted Text"
- terminal support affects UI quality
- AI may roleplay model identity

## Serena Gotchas
- Gemini may hallucinate Serena commands
- Serena must be validated against installed version
- indexing is not the same as deep architecture review

## File State Beats Chat Memory
If uncertain, inspect the repository.
