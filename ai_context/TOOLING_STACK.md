# Tooling Stack

## Primary Tools

### ChatGPT
Used for:
- strategy
- architecture
- review
- continuity
- high-value reasoning

### Gemini CLI
Used for:
- repo work
- shell interaction
- implementation
- source review

### Serena
Used for:
- repo-aware inspection
- project indexing
- symbol / structure awareness

### Git
Used as deterministic work boundary and audit trail.

### Pytest / Validation Commands
Used to prove actual behavior.

## Important Principle
Tools are not authority.
Tools assist engineering discipline.
