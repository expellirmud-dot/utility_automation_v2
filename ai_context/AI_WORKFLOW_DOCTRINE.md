# AI Workflow Doctrine

## Strategic Role Split

### ChatGPT / GPT
Role:
- architect
- reviewer
- governance interpreter
- workflow designer
- escalation authority
- second-opinion reviewer

Best used for:
- architecture arbitration
- deep reasoning
- governance review
- doctrine design
- prompt / workflow design
- critical review

### Gemini CLI
Role:
- repo operator
- implementation worker
- shell-assisted coding assistant
- inspection executor

Best used for:
- repo traversal
- READ-FIRST inspection
- implementation tasks
- test scaffolding
- source-level review
- practical engineering work

### Serena
Role:
- repository intelligence
- structural inspection
- symbol-aware context
- repo reality anchor

## Operating Principle
GPT thinks at governance / architecture level.
Gemini operates inside the repository.
Serena grounds analysis in actual code.
